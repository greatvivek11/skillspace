# SkillSpace Portfolio Copy

## Website Blurb

SkillSpace is a semantic resume-to-job matching demo built for recruiter-style workflows. It uses sentence-transformer embeddings, FAISS retrieval, and deterministic skill-gap reasoning to turn an uploaded candidate resume into ranked job matches and explainable skill insights.

Live demo: [greatvivek11/skillspace-demo](https://huggingface.co/spaces/greatvivek11/skillspace-demo)

## Short Website Version

Built and deployed a semantic resume-matching app that maps candidate resumes to likely-fit roles using embeddings, vector search, and explainable skill-gap logic.

## Resume Bullets

- Built and deployed `SkillSpace`, a public Hugging Face Space that matches candidate resumes to relevant roles using sentence-transformer embeddings, FAISS retrieval, and deterministic skill-gap analysis.
- Designed an end-to-end ML product slice including public dataset ingestion, schema normalization, embedding generation, retrieval evaluation, and a recruiter-facing Gradio UI.
- Published supporting dataset artifacts and deployment-ready app infrastructure on Hugging Face for a portfolio-ready semantic search demo.
